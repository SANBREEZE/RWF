#### Responsive Web JavaScript

The SANBREEZE Responsive Web JavaScript framework implements the repsonsive web-layout to any website.

---

#### Requirements  :
  jQuery 1.11.0 or higher (except 2.x or higher)

---

#### Usage:
###### Standard Usage:
``` javascript
$().SB_RWJS().setup();
```

###### Optional Usage:
```javascript
$().SB_RWJS().setup({
});
```
